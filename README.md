# 🎓 مشاوره نوین - سایت با پنل مدیریت

```

├── index.html          ← صفحه اصلی سایت
```

## 🚀 مراحل راه‌اندازی (فقط ۵ قدم!)

### قدم ۱: ساخت حساب GitHub
1. برو به [github.com](https://github.com)
2. روی **Sign up** بزن
3. با ایمیل ثبت‌نام کن
4. یه **Repository** جدید بساز (مثلاً `moshaverehnovin`)

### قدم ۲: آپلود فایل‌ها
1. وارد Repository شو
2. روی **Add file** → **Upload files** بزن
3. همه فایل‌ها و پوشه‌ها رو آپلود کن
4. روی **Commit changes** بزن

### قدم ۳: اتصال به Netlify
1. برو به [netlify.com](https://netlify.com)
2. با همون ایمیل GitHub وارد شو
3. روی **Add new site** → **Import an existing project** بزن
4. **GitHub** رو انتخاب کن
5. Repository `moshaverehnovin` رو پیدا و انتخاب کن
6. روی **Deploy site** بزن

### قدم ۴: فعال‌سازی پنل مدیریت
1. توی داشبورد Netlify، تب **Identity** رو باز کن
2. روی **Enable Identity** بزن
3. تب **Registration** → **Invite only** انتخاب کن
4. تب **Services** → **Git Gateway** → **Enable Git Gateway** بزن

### قدم ۵: ورود به پنل مدیریت
1. آدرس سایتت رو باز کن: `https://your-site.netlify.app`
2. به آدرس `/admin/` برو: `https://your-site.netlify.app/admin/`
3. روی **Sign up** بزن و با ایمیل ثبت‌نام کن
4. ایمیل تأیید رو چک کن و لینک رو باز کن
5. حالا می‌تونی وارد پنل مدیریت بشی! 🎉

## 📝 چطور محتوا رو ویرایش کنم؟

### از طریق پنل مدیریت (راحت‌ترین راه)
1. برو به `https://your-site.netlify.app/admin/`
2. وارد شو
3. از منوی سمت چپ بخش مورد نظر رو انتخاب کن:
   - 🔥 **Hero** → ویرایش عنوان، توضیحات، دکمه، آمار
   - 📚 **خدمات** → اضافه/حذف/ویرایش خدمات
   - ⭐ **نظرات** → اضافه/حذف نظرات دانش‌آموزان
   - 🎓 **کارنامه‌ها** → اضافه/حذف کارنامه‌ها
   - 📝 **مقالات** → اضافه/حذف مقالات
   - ❓ **سوالات متداول** → اضافه/حذف FAQ
   - ⚙️ **تنظیمات** → تاریخ کنکور، شماره تماس، ایمیل
4. تغییرات رو ذخیره کن (Publish)
5. خودکار سایت آپدیت میشه! ✨

### از طریق ویرایش فایل JSON (برای کاربر حرفه‌ای)
می‌تونی مستقیم فایل‌های `content/*.json` رو توی GitHub ویرایش کنی.

## 🆘 مشکل دارید؟

| مشکل | راه‌حل |
|------|--------|
| نمی‌تونم وارد پنل بشم | Identity → Registration → Invite only باشه |
| تغییرات اعمال نمیشه | ۲ دقیقه صبر کن، بعد Ctrl+F5 بزن |
| تصویر آپلود نمیشه | Git Gateway فعال باشه |
